# SQLScripter



Generador de scripts en masa para archivos SQL compatibles con SQL Server 2016 o superiores.



## Requisitos



* .NET Framework 4.7.1 o superior

* Visual Studio 2017 o superior



## Instrucciones de uso



#### Acceso a clases 



1. Abrir solución en VS

2. Restaurar paquetes NuGet de la solución

3. Compilar el proyecto como "Release"

4. Agregar el dll generado como referencia al proyecto que lo necesite



#### Demostración


1. Abrir solución en VS

2. Restaurar paquetes NuGet de la solución

3. Compilar el proyecto como "Release"

4. Abrir el archivo .exe generado



## Construido con



* SQL Server Management Objects - Creación de scripts individuales

* SharZipLib - Generación de archivos comprimidos

* Spire.XLS - Procesamiento de documentos
